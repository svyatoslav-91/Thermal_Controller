Разработанное устройство (включая всю требуемую документацию согласно ЕСКД) — ПИД-регулятор контроля температуры печи оплавления с программируемым термопрофилем. 
Устройство снабжено преобразователем напряжения, подключающимся к бытовой электрической сети 230 В. Преобразователь питания снабжает энергией как цифровую часть устройства управления, так и аналоговую часть нагревательного устройства и устройства контроля температуры. 
Устройство имеет интерфейс взаимодействия с пользователем (ЖК-дисплей и клавиатурную панель). Также разработано ПО для управления, настройки и тестирования устройства с ПК на языке С. 
Цифровая часть управления реализована на микроконтроллере с ARM-архитектурой STM32F411CEU6 — микроконтроллер, 32-бит, Cortex-M4, 100 МГц, 512 КБ Flash, 128 КБ SRAM, 36 I/O [UFQFPN-48_7x7]. 
Аналоговые устройства взаимодействия с нагревательным элементом гальванически развязаны. Термопара в связке с MAX6675ISA по SPI передает данные о температуре в МК, который с помощью ПИД регуляции вычисляет необходимое время включения нагревателя в окне регулирования. 
Коммутирование нагревателя осуществляюется с помощью SSR реле что обеспечивает полную гальваническую развязку от цепи 220 В нагревателя, а также быстрый отклик.

По ссылке на видео размещена демонстрация работы устройства: программирование термопрофиля, запуск ПИД-регулятора, отображение температуры на ЖК-дисплее и в ПО на ПК, а также реакция системы на изменение уставки, запуск различных термопрофилей и запись во flash. (в качестве нагревателя для удобства используется лампочка накаливания).

В корневой папке проекта можно найти:

принципиальные электрические схемы (в форматах PDFисходники для Altium Designer);

Gerber-файлы для изготовления печатной платы;

сборочные чертежи платы и корпуса (в форматах PDF);

спецификации компонентов (ВП, ПЭ3)(в форматах PDF);

3D-модель печатной платы (Step) для проверки совместимости с корпусом и механической интеграции;

исходные коды встроенного ПО для STM32F411CEU6 (с проектом для Keil/IAR/STM32CubeIDE);

исходный код ПО для ПК на языке C;

Вся конструкторская документация оформлена по стандартам ЕСКД (схемы, чертежи, спецификации, пояснительная записка). Проект полностью готов к серийному производству и может быть воспроизведён по предоставленным материалам.

В release-пакет включены:

исполняемый файл приложения для ПК (Windows) для управления, настройки и тестирования устройства;

краткое руководство пользователя (Release Notes).
